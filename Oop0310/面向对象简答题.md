```
简答题
1.什么是封装
2.static关键字的特点
3.&&和&  ||和|的区别
4.一个".java"源文件中是否可以包括多个类（不是内部类）？有什么限制？
5.什么叫方法的重写
6.静态方法,静态代码块,构造函数代码块,构造代码块执行先后顺序
```
```
答:
```
```
1.封装就是将属性私有，用来保护类中的数据，外界不能直接访问到你的属性，而是通过公有的方法来访问这个类。
```
```
2.  class文件只会被加载一遍
    class文件在内存中夹在的时候，被static修饰的成员立即加载 static修饰的成员叫 类成员
```
```
3.  "&" 如果两边为数字时，作为位运算，如果两边为表达式 表示逻辑判断。
    如果前面的表达式为假，后面还会继续判断。
    "&&" 表达式1 && 表达式2
    只有两个表达式都成立才为真（true）。
    如果第一个表达式为假，那么后面的表达式不会继续判断。
    "||" 短路或 表达式只要有一个成立就成立
    "|" true |判断
```
```
4.这个是可以的，一个“.java”源文件里面可以包含多个类，但是只允许有一个public类，并且类名必须和文件名一致。每个编译单元只能有一个public 类。这么做的意思是，每个编译单元只能有一个公开的接口，而这个接口就由其public 类来表示。
你可以根据需要，往这个文件里面添加任意多个提供辅助功能的package 权限的类。但是如果这个编译单元里面有两个或两个以上的public类的话，程序就不知道从哪里导入了，编译器就会报错。　　
```
```
5.在Java和其他一些高级面向对象的编程语言中，子类可继承父类中的方法，而不需要重新编写相同的方法。但有时子类并不想原封不动地继承父类的方法，而是想作一定的修改，这就需要采用方法的重写。方法重写又称方法覆盖。
```
